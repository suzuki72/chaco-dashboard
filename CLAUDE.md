# マーケティング部

## 部門概要

デザイン＆ランゲージ株式会社のマーケティング部門。
LINE スタンプ・絵文字・着せ替えの販売データ分析、SNS戦略、キャンペーン企画を担当する。

## ミッション

chacoのデザイン資産を最大限に活用し、スタンプ販売を軸にキャラクターの認知拡大と収益最大化を実現する。

## 事業の基本情報

- **主力商品**: LINE スタンプ、絵文字、着せ替え（chacoデザイン）
- **累計売上**: 1億円超
- **LINE公式アカウント**: 約80万フォロワー
- **主要キャラクター**（売上順）: パンダ > 柴犬 > 猫 > くますけ > ひつじ > ハムスター > うさこ
- **収益構造**: プレミアム（使い放題）約66%、通常販売 約34%（2026年2月時点）
- **海外比率**: 約0.6%（成長余地大。台湾・米国が有望）

## 戦略方針

1. **プレミアム収益を基盤としつつ、通常販売を追求する**
2. **スタンプ以外のチャネルでキャラクターを押し出し、スタンプ販売へ還元する**
   - LINE VOOM（ショート動画・画像）
   - その他 SNS（Instagram, X 等）
   - YouTube BGMチャンネル
   - LINE LIFF を使ったゲーム（キャラ連動）
   - スマホアプリ（chacoメモ等）
3. **ひつじ（Sheep）系を来年の干支（未年）に向けて重点強化**
4. **海外比率を高める**（台湾・米国を優先市場とする）

## 利用可能なツール・スクリプト

### 月次販売レポート生成

```bash
python3 ~/Documents/company-ai/marketing/line_stamp_monthly_report.py <CSVファイルパス>
```

- LINE Creators Market からダウンロードしたCSVを入力
- ターミナルに分析結果を表示し、同ディレクトリに txt レポートを保存
- カスタマイズポイント: `character_keywords` 辞書にキャラ名を追加可能

### CSVの構造メモ

| カラム | 意味 |
|--------|------|
| Revenue Share Rate = 35.0 | 通常販売 |
| Revenue Share Rate = 20.0 | 販売（低レート） |
| Revenue Share Rate = 空欄 | プレミアム（使い放題分配金） |
| Type: Sticker / Emoji / Theme | スタンプ / 絵文字 / 着せ替え |
| Compensation = N | 通常行（Yは補償関連） |

2026-03-23にリリース。LINE公式アカウントから配信。

### 分析ツール一覧

| ツール | 用途 | URL |
|--------|------|-----|
| Firebase Console | iOS リアルタイム & 日次指標 | console.firebase.google.com（プロジェクト: chaco-memo） |
| Google Analytics | 集客経路・リテンション詳細分析 | analytics.google.com（プロパティ: a50253662p526208629） |
| Google Play Console | Android インストール・DAU | play.google.com/console |
| AdMob | アプリ内広告収益 | admob.google.com |

### 主要KPI

- **first_open**（Firebase）: インストール→初回起動数
- **DAU / MAU**: 日次・月次アクティブユーザー
- **Day1 / Day7 リテンション**: 継続利用率
- **eCPM / インプレッション数**（AdMob）: 広告収益指標

### ローンチ初日ベースライン（2026-03-23）

| 指標 | 値 |
|------|-----|
| first_open | 689 |
| DAU | 660 |
| 平均エンゲージメント時間 | 2分11秒 |
| エンゲージメント率 | 0.98 |
| AdMob 推定収益（初日） | ¥183（iOS ¥132 / Android ¥52） |
| eCPM 平均 | ¥76 |
| 広告CTR | 5.50%（要注意：誤タップ疑い） |

### AdMob 運用上の注意

- **CTRが1%を超えたら広告配置を疑う**（正常値は0.3〜1%）。キーボードやボタン付近のバナーは誤タップを誘発しやすく、AdMobの低品質判定・収益剥奪リスクがある
- **eCPMの低さは初日は正常**。ターゲティングデータが蓄積される2〜3週間後に再評価する
- Firebase・AdMobともに **24〜48時間のデータ反映ラグ**があるため、当日中に数字が動かなくても正常

### LINE公式アカウントの実効リーチ

- 登録フォロワー約80万のうち、**ブロック等を除いた有効リーチは約30万**（2026-03時点）
- アプリ配信時の転換率の分母はこの30万で計算する（0.23% = 689/300,000）

## 定常業務
